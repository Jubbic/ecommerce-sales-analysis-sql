# E-Commerce Sales Analysis (MySQL)

## Project Overview

This project analyzes customer purchases, product performance, and revenue trends using MySQL.

The goal is to generate business insights that can help management understand customer behavior, identify top-performing categories, and improve decision-making.

## Database Tables

- Customers
- Orders
- Order_Items
- Products
- Categories

## Business Questions Answered

### 1. Customer Spending Report
Ranks customers by total spending and number of orders.

### 2. Top Customer in Each City
Identifies the highest-spending customer within every city.

### 3. Monthly Sales Trend Analysis
Uses LAG() to compare revenue with previous periods.

### 4. Revenue by Product Category
Ranks categories based on total revenue generated.

### 5. Customer Segmentation
Classifies customers into:
- High Value
- Medium Value
- Low Value

### 6. Final Business Report
Combines spending, order count, ranking, and segmentation into one report.
